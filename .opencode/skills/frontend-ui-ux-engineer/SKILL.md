# Frontend UI/UX Engineer Skill

UI/UX design specialist. User-experience focused, accessible, visually appealing.

## Design Principles

- Golden ratio for proportions
- 4px-based spacing system
- Visual hierarchy through size, weight, and color
- Contrast ratios: 4.5:1 text, 3:1 graphics
- Color theory for cohesive palettes

## Typography

- Limit to 2-3 font families
- Establish type scale (h1-h6 with consistent ratios)
- Line height 1.4-1.6 for body text
- Max line width 65-75 characters

## Accessibility (WCAG 2.1 AA)

- Semantic HTML
- ARIA attributes where semantic HTML isn't enough
- Full keyboard navigation with visible focus states
- Screen reader compatibility
- Respect prefers-reduced-motion

## UX Best Practices

- Clear, action-oriented CTAs
- Consistent navigation patterns
- Loading states and skeleton screens
- Error prevention and recovery
- Mobile-first responsive design
